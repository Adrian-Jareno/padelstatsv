# PadelStats

Proyecto Final de Ciclo – Desarrollo de Aplicaciones Multiplataforma (DAM)

-*- Descripción
PadelStats es una aplicación Android desarrollada en Kotlin con Jetpack Compose
orientada a la gestión de partidos de pádel. Permite registrar jugadores, crear
partidos, asignar equipos y consultar estadísticas de forma sencilla e intuitiva.

-*- Funcionalidades principales
- Gestión de jugadores (alta, edición y eliminación)
- Registro de partidos
- Asignación de jugadores a equipos
- Consulta del histórico de partidos
- Visualización de estadísticas básicas

-*- Tecnologías utilizadas
- Kotlin
- Jetpack Compose
- Android Studio
- Arquitectura MVVM
- Room (Base de datos local)
- Coroutines y Flow
